# Calendar Overview

The calendar is your command center for managing installations and appointments.

## Views

Switch between views using the buttons at the top:

- **Day** - Detailed view of a single day
- **Week** - See the full week at a glance
- **Month** - Monthly overview

## Bays and Technicians

The calendar shows:

- **Bays** - Your installation bays (columns)
- **Appointments** - Scheduled work (blocks on the calendar)
- **Availability** - Open time slots

## Quick Actions

- **Click empty slot** - Create new appointment
- **Drag appointment** - Reschedule
- **Click appointment** - View details

## Color Coding

Appointments are color-coded by status:

| Color | Status |
|-------|--------|
| Blue | Scheduled |
| Yellow | In Progress |
| Green | Completed |
| Red | Overdue |
