# Creating Quotes

Quotes are the starting point for every job in Shop Director.

## Overview

The quote workflow follows this path:

**Quote → Order → Work Order → Invoice**

This guide covers creating and managing quotes.

## Creating a New Quote

1. Navigate to **Quotes** in the sidebar
2. Click **New Quote**
3. Add customer and vehicle information
4. Add line items (products and services)
5. Save or send to customer

<!-- SCREENSHOT: /quotes/new | New quote form | highlight:input[type="submit"],.btn-primary -->

## Quote Statuses

| Status | Meaning |
|--------|---------|
| Draft | Work in progress, not sent |
| Sent | Emailed to customer |
| Approved | Customer accepted |
| Declined | Customer declined |
| Expired | Past expiration date |

## Converting to an Order

Once a quote is approved:

1. Open the quote
2. Click **Convert to Order**
3. The quote becomes an order ready for scheduling
