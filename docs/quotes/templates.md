# Quote Templates

Save time by creating templates for common jobs.

## What Are Templates?

Templates are pre-configured quotes with common products and services. Use them as starting points for new quotes.

## Creating a Template

1. Create a quote with your common line items
2. Click **Save as Template**
3. Give it a descriptive name

<!-- SCREENSHOT: /quotes/1 | Save as template button | highlight:a[href*="template"],.dropdown-item -->

## Using a Template

1. Start a new quote
2. Click **From Template**
3. Select your template
4. Customize as needed

## Template Ideas

- Remote start installation (basic)
- Remote start installation (premium)
- Car audio upgrade package
- Security system installation
- LED lighting package
